# KRYPSIS Forensic Suite  🛡️🔍

**KRYPSIS** is a high-performance, modular digital forensics ecosystem designed for rapid evidence analysis. It bridges the gap between raw data collection and actionable intelligence through a specialized multi-module investigative workflow.

---

## 🌟 Overview

In modern investigations, speed and accuracy are paramount. **KRYPSIS v6.0** allows investigators to ingest digital evidence and pivot between specialized analysis modules, ensuring a comprehensive 360-degree forensic review while maintaining a strict chain of custody.

---

## 🚀 Key Investigative Workflow

* **Evidence Ingestion:** Centralized image/data upload system for evidence entry.
* **Multi-Module Analysis:** Specialized drill-down modules for deeper investigation into different forensic categories.
* **Autonomous Integrity Checks:** Real-time hashing (SHA-256) during every analysis step to prevent data tampering.
* **Deep-Dive Scanning:** Dedicated interfaces for metadata recovery, log parsing, and encrypted fragment analysis.
* **Professional Forensic UI:** High-contrast, no-distraction Dark Mode optimized for tactical environments.

---

## 🛠️ Technical Stack

KRYPSIS is built with a modern, modular architecture for scalability:

-   **Frontend:** React.js (v18+) & Vite
-   **Styling:** Tailwind CSS & Shadcn/UI
-   **Architecture:** Component-based modular design for different forensic tasks.
-   **State Management:** TanStack Query for seamless data transitions between modules.
-   **Iconography:** Lucide-React & Custom Forensic Symbols.

## 📦 Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/](https://github.com/)[YOUR_USERNAME]/Krypsis-forensics-suite.git
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

## 📂 Project Structure

-   `/src/pages`: Contains the specialized Analysis Modules (Evidence Vault, Metadata, etc.).
-   `/src/components`: UI elements like forensic grids, sidebar, and uploaders.
-   `/public`: Houses the KRYPSIS identity assets.
